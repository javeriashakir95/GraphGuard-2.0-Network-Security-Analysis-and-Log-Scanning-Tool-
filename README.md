# GraphGuard-2.0-Network-Security-Analysis-and-Log-Scanning-Tool-

## Overview

Graph Guard 2.0 is a Network Security Analysis and Log Scanning Tool developed as a Data Structures and Algorithms (DSA) project. The system helps analyze network structures, identify potential security threats, and visualize network connections through an interactive graphical interface.

The project demonstrates the practical application of DSA concepts in cybersecurity by combining graph algorithms, network analysis, and log scanning techniques.

---

## Features

* Network topology visualization using graph data structures
* Shortest path analysis using Dijkstra's Algorithm
* Network traversal using Breadth-First Search (BFS)
* Security log scanning and analysis
* Threat detection using Regular Expressions (Regex)
* Alert generation for suspicious activities
* Interactive graphical user interface (GUI)
* Security analytics and reporting
---

## Data Structures & Algorithms

### Data Structures

* Graphs (Adjacency Lists)
* Hash Tables (Dictionaries)
* Queues
* Priority Queues (Heaps)

### Algorithms

* Dijkstra's Shortest Path Algorithm
* Breadth-First Search (BFS)
* Pattern Matching using Regular Expressions

## Project Objectives

* Apply Data Structures and Algorithms to solve real-world problems.
* Visualize and analyze network connections efficiently.
* Detect potential security threats from network logs.
* Develop an easy-to-use interface for network monitoring and analysis.
* Enhance understanding of graph-based problem solving.
